# School-Database
# School Database Management System

A simple MySQL project that demonstrates database design and SQL operations.

## Features
- Create School Database
- Create Students, Teachers, and Subjects tables
- Insert sample records
- Update records
- Rename tables
- Create a new table using SELECT
- Sorting and filtering data

## Technologies
- MySQL
- SQL

## Concepts Used
- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- UPDATE
- ALTER TABLE
- CREATE TABLE AS SELECT
- ORDER BY
- WHERE
- LIKE
